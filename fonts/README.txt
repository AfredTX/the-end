TOP-LEFT ANCHOR
===============

The top-left "THE END." mark is an IMAGE.

Drop your image into the PROJECT ROOT (one level up from this folder),
named exactly:

    the-end.png

That's it — index.html already points to it. Until the file exists,
the site shows a "THE END." text fallback styled in the Loubag font
(or a serif if Loubag isn't installed).


LOUBAG FONT (only the text fallback uses this — optional)
=========================================================
If you ever want the *text* fallback to render in Loubag:

1. Get the Loubag font files.
2. Export to web formats (.woff2 preferred). Free converter:
   https://www.fontsquirrel.com/tools/webfont-generator
3. Put them in THIS folder named:
     Loubag-Bold.woff2 / Loubag-Bold.woff
